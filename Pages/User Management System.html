User Management System

Description

I developed a User Management System using full CRUD operations where administrators can add, update, delete, and view user information. The project is built with Node.js, Express.js, MongoDB, and a front-end using HTML, CSS, and JavaScript.

Features

The system includes secure login and registration features, and uses password hashing for safety. Admins have access to manage all users, while users can view or edit their own profiles. Role-based access ensures the right level of control for each user type.

This project helped me strengthen my understanding of backend APIs, MongoDB integration, and secure authentication, all while delivering a responsive and clean user interface.
