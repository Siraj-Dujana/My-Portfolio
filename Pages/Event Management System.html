Event Management System

Description:

This project is all about making event planning easier. Whether it's a tech meetup or a wedding, users can register for events, and admins can create, update, and manage them from a dashboard. I added user login, event details pages, and full CRUD functionality for events. It’s got a clean UI and works well on mobile too. The backend handles everything from user sessions to storing event data securely in MySQL.

Features:


Event CRUD Operations: Admins can create, update, and delete events with full control over event details like date, time, and location.

Admin Dashboard: A dedicated backend panel for admins to view all events, manage users, and monitor registrations.

Event Registration: Logged-in users can browse available events and register with a single click. Registrations are tracked in real-time.

Responsive Design: The UI works seamlessly across desktop, tablet, and mobile devices for a smooth user experience.

Secure Data Handling: Inputs are validated, and sessions are protected to prevent unauthorized access or data leaks.
