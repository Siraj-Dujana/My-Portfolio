User Management System

Developed a full-featured User Management System that enables the creation, reading, updating, and deletion of user data (CRUD). This project was built using Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript.

The system allows users to register and log in securely, with hashed passwords and session-based authentication. Admins have access to additional features, including the ability to manage all users in the system. Each user has a profile that can be updated or deleted, with role-based access control implemented for better security.

CRUD operations are implemented on the backend using Express.js APIs, with MongoDB as the database for scalable and efficient user data handling. The UI is clean and responsive, making the platform usable on both desktop and mobile devices.

This project is a solid example of how to manage user data securely and efficiently in a full-stack application while providing a smooth and functional user interface.

Recipe Sharing App

Developed a full-stack Recipe Sharing App that allows users to share, view, and search for delicious recipes. Built using Node.js, Express.js, MySQL, HTML, CSS, and JavaScript, the app provides an intuitive and user-friendly platform for food enthusiasts to discover and share recipes.

The app features a user-friendly interface that lets users browse a variety of recipes by category, cuisine, or ingredients. Users can submit their own recipes, which include detailed instructions, ingredients, and photos. The recipe submission form also includes real-time validation to ensure the data is entered correctly.

To store and manage recipe data, a robust back-end was built using Node.js and Express.js APIs. The app interacts with a MySQL database for secure and scalable data storage. Implemented CRUD operations to create, read, update, and delete recipes, as well as user authentication to ensure secure logins and personalized recipe management.

The front-end was designed with HTML and CSS, ensuring a responsive layout that works seamlessly across mobile and desktop devices. Enhanced interactivity was added using JavaScript for a smooth user experience, including AJAX calls for dynamic content updates without page reloads.

This project showcases my ability to build full-stack applications, manage databases, and create a visually appealing and functional front-end, making it an excellent addition to my development portfolio.
